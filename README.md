# Diversity Duel

A single-player web game that teaches AI ethics through prompt engineering. Write prompts under shrinking word limits to generate the most diverse AI images possible — and discover how every word shapes what AI produces.

**Play:** [diversity-duel.tandemly.ai](https://diversity-duel.tandemly.ai)

## How It Works

1. Draw a random occupation card (e.g., "construction workers")
2. Write a prompt for an AI image generator under a word limit
3. AI judges your prompt across 5 diversity dimensions
4. Word limit shrinks each round: 6 → 5 → 4 words
5. Learn how prompt wording reproduces or challenges AI bias

## Based On

This game adapts the research from:

> Solyst, J., Nakigozi, R. K., Fong, C., & Shapiro, R. B. (2025). "Designing Transformational Games to Support Socio-ethical Reasoning about Generative AI." International Society of the Learning Sciences. arXiv: 2604.02154v1.

Read the synthesis: [Games That Teach AI Ethics](https://tandemly.ai/research/transformational-games)
